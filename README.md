# ParkIQ 🚗

ParkIQ is a Smart Parking Management System built using Flask, Python, HTML, CSS, JavaScript, and SQLite.

## Features
- Real-time parking search by location
- GPS-based location detection
- AI parking availability prediction
- Slot booking system
- Dynamic pricing calculation
- Razorpay payment integration
- PDF receipt generation
- Parking timer countdown
- SMS expiry alerts
- Mobile responsive design

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python Flask

Database:
- SQLite

APIs Used:
- Razorpay
- OpenStreetMap
- Nominatim
- SMS API
